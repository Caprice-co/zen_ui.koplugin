## Overview

Global settings apply across KOReader. They cover whole-word search, night mode scheduling, frontlight brightness scheduling, warmth scheduling, sleep controls, and Lockdown Mode configuration.

## Options

- Set search to match whole words instead of substrings.
- Schedule night mode on and off times.
- Schedule day and night brightness levels.
- Schedule day and night warmth levels on devices with natural light support.
- Access KOReader sleep screen controls, sleep presets, automatic dimmer, and automatic suspend integrations when available.
- Configure Lockdown Mode restrictions for library, Controls, and reader interactions.

## Setting reference

| Setting | Description |
| --- | --- |
| Search > Match whole words | Uses whole-word search instead of substring search. |
| Night mode schedule > Enable night mode schedule | Enables automatic night mode changes. |
| Night mode schedule > Night mode on | Sets the time night mode turns on. |
| Night mode schedule > Night mode off | Sets the time night mode turns off. |
| Brightness schedule > Enable brightness schedule | Enables automatic frontlight brightness changes. |
| Brightness schedule > Day brightness time | Sets the daytime brightness schedule time. |
| Brightness schedule > Day brightness | Sets the daytime brightness value within the device range. |
| Brightness schedule > Night brightness time | Sets the nighttime brightness schedule time. |
| Brightness schedule > Night brightness | Sets the nighttime brightness value within the device range. |
| Warmth schedule > Enable warmth schedule | Enables automatic warmth changes on natural-light devices. |
| Warmth schedule > Day warmth time | Sets the daytime warmth schedule time. |
| Warmth schedule > Day warmth | Sets the daytime warmth value within the device range. |
| Warmth schedule > Night warmth time | Sets the nighttime warmth schedule time. |
| Warmth schedule > Night warmth | Sets the nighttime warmth value within the device range. |
| Sleep > KOReader screensaver controls | Shows supported KOReader sleep screen options when the device supports screensavers. |
| Sleep > Presets | Applies, saves, and deletes sleep screen presets. Built-in presets include Book cover - Black Fill, Zen - White, Zen - Transparent, and Zen - Black. |
| Sleep > Automatic dimmer | Shows autodim controls when the autodim plugin is available. |
| Sleep > Automatic suspend | Shows autosuspend, autoshutdown, and autostandby controls when the autosuspend plugin is available. |
| Lockdown mode > Magnify UI | Enlarges library layout density while Lockdown Mode is active. |
| Lockdown mode > Library | Configures library context-menu restrictions. |
| Lockdown mode > Controls | Configures hold-to-toggle and settings-panel restrictions. |
| Lockdown mode > Reader | Configures reader bottom swipe, multi-word selection, and hold search restrictions. |
